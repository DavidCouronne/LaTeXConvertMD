# Syntaxe de base

## Maths en ligne

```md
$\sqrt{2}$
```

$\sqrt{2}$

::: warnig Attention
+ Il faut bien "coller" les $ à l'expression
+ Impossible de "passer à la ligne
:::

Exemples:

``` md
$\sqrt{2} $
```

$\sqrt{2} $

``` md
$f(x)=(x+3)^2
=x^2+6x+9$
```

$f(x)=(x+3)^2
=x^2+6x+9$
